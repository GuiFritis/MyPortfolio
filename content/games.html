Jogos

Underground

Esse jogo foi desenvolvido em uma semana para a Brackeys Game Jam, o game design e a programação foram totalmente feitos por mim usando Unity.
Os sprites e audio eu usei da Unity Asset Store.

Underground is a game where the player is in a cave going deeper underground, and by doing so worlds are uncovered showing that there is more underneath the surface.

The genre of the game is platform roguelite, so you go as far as you can and repeat, every run being randomly generated, your goal is to reach as far as you can.

Maze Game

This game was made with the playlist "Making Your First Game in Unreal Engine 4" from Ryan Laley, using the assets and following the steps in these tutorials with some aditional features of my own.

In the game you are in a maze and must find the golden platform to escape, avoiding traps and using keys to unlock your way.

Astronault vs Slimes 2D

This game I made in my Unity development course with the help of the instructor. Though, some of the developments and ideas came from me.

Astronault vs Slimes 2D is a plaform adventure game where, as the game sugests, you control an astronault fighting your way against slimes while colleting coins.

Ship Wrecking

I made all of the development in the game, the art and inicial concept was given to me, though I gave some additional features to the game.

Ship Wrecking is a roguelite game where you are a ship and you must survive and sink other ships in order to gain points.

Astronault vs Slimes 3D

This game I also made in my Unity development course with the help of the instructor. I've put a lot of original features in the game as it was my final game for the course.

Astronault vs Slimes is a 3D adventure-shotting game that we can call a continuation from the 2D version.

In this game you can scroll through 2 types of weapons each with it's own ammo and recharge rate. The goal of the game is to kill the boss in the end, and collect as many coins as you can.
